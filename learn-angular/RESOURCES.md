# Angular Resources

## Knowledge

- [Angular Official Documentation — angular.dev/guide/](https://angular.dev/guide/)
  Comprehensive guide covering all Angular concepts, from basics to advanced. Use for: primary learning source, understanding core concepts, modern Angular patterns.
- [Angular Style Guide — angular.dev/guide/style-guide](https://angular.dev/guide/style-guide)
  Official recommendations for Angular project structure and naming conventions. Use for: architectural decisions, code organization.
- [Angular Blog — blog.angular.io](https://blog.angular.io/)
  Official Angular team blog with updates, deep dives, and case studies. Use for: staying current with latest features and best practices.
- [Angular University — angular-university.net](https://angular-university.net/)
  High-quality video courses and tutorials by Lucas Paganelli. Use for: visual learners, practical examples.
- [NgConf Talks — youtube.com/@angular](https://www.youtube.com/@angular)
  Official Angular YouTube channel with conference talks and tutorials. Use for: learning from core team and experts.

## Wisdom (Communities)

- [r/Angular2 — reddit.com/r/Angular2](https://www.reddit.com/r/Angular2/)
  Active Angular community for questions, discussions, and help. Use for: troubleshooting, getting diverse perspectives.
- [Angular Discord — discord.gg/angular](https://discord.gg/angular)
  Official Angular Discord server for real-time help and discussions. Use for: quick questions, community interaction.
- [Stack Overflow — angular tag](https://stackoverflow.com/questions/tagged/angular)
  Question and answer site with extensive Angular knowledge base. Use for: specific technical problems.
